## 📐 图片规格要求

### 封面图统一规范
| 类型 | 尺寸 | 格式 | 最大大小 |
|------|------|------|----------|
| 笔记封面 | 800x600 (4:3) | JPG/PNG | 300KB |
| 文章封面 | 1200x630 (1.91:1) | JPG/PNG | 500KB |
| 头像 | 500x500 (1:1) | JPG/PNG | 200KB |
| Logo | 200x200 (1:1) | PNG(透明) | 100KB |
| 二维码 | 500x500 (1:1) | PNG(透明) | 200KB |

### 图片优化建议
1. **压缩工具**：TinyPNG、ImageOptim
2. **WebP格式**：支持浏览器可转换为WebP
3. **懒加载**：使用Next.js Image组件
4. **CDN**：生产环境使用CDN加速

## 🔧 批量处理脚本

### 使用ImageMagick批量调整尺寸
```bash
# 批量调整笔记封面为800x600
mogrify -resize 800x600! -quality 85 *.jpg

# 批量转换为WebP
mogrify -format webp *.jpg
```

### 使用ffmpeg批量压缩
```bash
# 批量压缩图片
for file in *.jpg; do
  ffmpeg -i "$file" -quality 85 "compressed_$file"
done
```
