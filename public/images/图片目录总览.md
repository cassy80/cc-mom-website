## 🛠️ 图片处理工具推荐

### 在线工具
- **TinyPNG**：https://tinypng.com/ - 图片压缩
- **Squoosh**：https://squoosh.app/ - 图片压缩和转换
- **Photopea**：https://www.photopea.com/ - 在线图片编辑器

### 本地工具
- **ImageOptim** (Mac) - 图片压缩
- **FileOptimizer** (Windows) - 图片压缩
- **Adobe Photoshop** - 专业图片编辑
- **GIMP** - 免费图片编辑器

### 批量处理
```bash
# 使用ImageMagick批量调整尺寸
mogrify -resize 800x600! -quality 85 *.jpg

# 批量压缩
for file in *.jpg; do
  convert "$file" -quality 85 "compressed_$file"
done
```
